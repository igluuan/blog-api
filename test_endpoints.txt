
### Create a new Post (POST /api/posts/new)
```json
{
  "title": "My First Blog Post",
  "content": "This is the content of my first blog post. It's very exciting!"
}
```

### Create a new Comment (POST /api/comments/new)
```json
{
  "content": "This is a great post!",
  "authorId": "a1b2c3d4-e5f6-7890-1234-567890abcdef",
  "postId": "12345"
}
```
